javaとはなにか

JAVAとは世界的に多く使用されているプログラミング言語のひとつです。
OSに依存せずWEBでの様々な機能などを作成することができます。
またこの言語を使ってゲーム作成などもできます。
(例)

  • <input type="button" value="画像の変更" onclick="imgchange01();">
  • <input type="button" value="戻る" onclick="imgchange02();">
  • <img src='./image/2.jpg' width="320" height="240" id="logo01" alt="run ">

ここまでできてもJAVAは動きません。そこでもう一つファイルをつくります。
  • function imgchange01() { document.getElementById('logo01').src = './image/1.png'; }
  • function imgchange02() { document.getElementById('logo01').src = './image/2.jpg'; }
この「getElementById」でHTMLを書き換えることができるので画像を切り替えることができます。
